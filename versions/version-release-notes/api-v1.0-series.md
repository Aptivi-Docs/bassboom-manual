# API v1.0 series

This series is the first version series that actually defines the multiplatform music player to the .NET environment. The first version of BassBoom, 0.0.1, was out on October 1st, 2023.

## Version 0.0.2

This version was released on November 14th, 2023, as a celebration of the .NET 8.0 release! This version serves as an enhancement release over the previous version.

### Improvements (Basolia)

* <mark style="color:yellow;">Updated MPG123 to 1.32.3</mark>
* <mark style="color:yellow;">Updated targeting to .NET 8.0</mark>
* <mark style="color:green;">Added support for macOS 64-bit (ARM64 support not done yet)</mark>
* <mark style="color:green;">Added getting frame length and samples per frame</mark>
* <mark style="color:green;">Added synthesis to Native (currently unused)</mark>

### Improvements (CLI music player)

* <mark style="color:yellow;">Updated Terminaux</mark>
* <mark style="color:yellow;">Show song name and duration in the music list</mark>
* <mark style="color:yellow;">Don't show genre in the titles</mark>
* <mark style="color:yellow;">Use music file name instead of "unknown song"</mark>
* <mark style="color:yellow;">Only populate music info if necessary</mark>
* <mark style="color:yellow;">You can remove songs from the playlist</mark>

### Improvements (GUI music player)

* <mark style="color:yellow;">Updated Avalonia</mark>
* <mark style="color:yellow;">Show song name and duration in the music list</mark>
